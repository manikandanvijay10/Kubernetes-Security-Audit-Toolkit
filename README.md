# Kubernetes-Security-Audit-Toolkit
A practical Kubernetes security audit, pentesting, and hardening toolkit for DevSecOps engineers and security professionals.
Includes:

• Kubernetes security checklist  
• Automated audit scripts  
• Sample pentest report  
• Hardening guide  
• Tool installation guide
